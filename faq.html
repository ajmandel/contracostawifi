FAQ

If you’re wondering it, other parents are too.

Do I need gigabit internet? ▾

Usually, no. Many homes run great on 100–300 Mbps. The bigger issue is Wi‑Fi setup inside the home.

Can I keep my ISP plan? ▾

Yes. We work with your ISP connection and improve your home network behind it.

Will you see my private data? ▾

We manage network settings, not your personal files. We also avoid collecting unnecessary data.

Can I still control my network? ▾

Yes. Co‑managed means you can view and control things too. We document everything.

What if something breaks? ▾

We troubleshoot remotely first. If we need to come out, we schedule quickly.

Is this safe for kids? ▾

That’s the point. We can set safe defaults, schedules, and device grouping.
